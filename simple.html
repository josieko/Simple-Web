HTML aka the 🦴

What is it

- HTML is a (HyperText) Markup language that is not really considered a coding language.
- Used to structure webpage content, as the skeleton, it defines how the elements should be displayed
- It is best known for the < > brackets that are used to define HTML elements

What is it good for

- It’s good for creating web page structures, email templates and online forms

What is it not good for

- It’s not good for styling web pages and that’s where CSS comes in, and the same thing with interactivity, where JavaScript is used to fill that gap
- HTML was intended to be a simple markup language with a focus on structuring and presenting content. - The simplicity of it is both a strength (for it being simple and accessible) and a weakness in terms of it not having capabilities for more complex tasks. That’s why it is used in combination with languages like CSS (to style and format HTML content), JS (to add interactivity), or even SQL (to process and manipulate data)

Who uses it

- It’s a widely spread language that anyone can use to create a web page, from professional web devs, content creators, and bloggers or it could be for a hobby

Companies that implement it

- Any company, business or individual having a website, that website uses HTML so it is pretty much a base for it all.
- Everyone uses it, from Google, Amazon and Apple to Netflix, Spotify, Wikipedia and the list goes on

Cool TikTok finds and tools

🐚 HTML5up - Website with free downloadable templates
- Emmet (integrated in VS Code) allows you to be faster by typing just the name of the element, by typing enter you get the opening and closing tags without manually writing them

Fun facts

- HTML is the oldest markup language on the web and it was invented by Tim Berners-Lee, the same guy who invented WWW - The world’s first web page was written in it



CSS aka the 👘

What is it

- Cascading Style Sheets (CSS) is a language used to style and describe how will the website be presented. That includes colors, fonts and layouts

What is it good for

- It’s good to style and control the look of the page, and this includes controlling the layout of the page, the appearance of text, images, links and other elements like tables, forms or buttons

What is it not good for

- Like HTML, CSS is used in combination with other languages to complement them, and it isn’t good for defining the structure of the page or interactivity.
- This is because it is primarily a styling language, not designed to handle complexities such as data manipulation or interactivity
- Although it is set to control layouts of pages, CSS can struggle with extremely complex animations (for example a 3D product viewer with options to rotate and zoom) and in cases like that, CSS might not perform well. That’s where specialized libraries come in like GSAP, Anime.js or TweenMax, and WebGL libraries like Three.js, Babylon.js or A-Frame
- All of the libraries from above can be used to create complex animations on mobiles as well.

Who uses it

- Just like HTML, anyone who makes or has a website, business or individual, will use CSS to style the appearance of their website
